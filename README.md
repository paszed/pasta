## ✨ What is Pasta?

Pasta is a fast, no-friction tool for cleaning messy text.

No setup. No login. Just paste and clean.

---

## ⚡ Features

- Removes duplicate lines  
- Trims whitespace  
- Removes empty lines  
- Cleans text instantly as you type  

---

## 🧠 Usage

1. Paste your text  
2. Get cleaned output instantly  
3. Copy and use  

---

## ⚡ Example

### Input:
```
john@test.com  
john@test.com  
   jane@test.com  

bob@test.com  
```

### Output:
```
john@test.com  
jane@test.com  
bob@test.com  
```
