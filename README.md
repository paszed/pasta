# 🍝 Pasta

**Paste messy text → get clean, usable output instantly.**

---

## ✨ What is Pasta?

Pasta is a simple browser-based tool that cleans up messy text data in seconds.

No setup. No login. Just paste and clean.

---

## 🚀 Features

- Remove duplicate lines
- Trim whitespace
- Remove empty lines
- Normalize formatting

---

## 🧠 Usage

1. Paste your messy text into the input box
2. Click **Clean**
3. Copy the cleaned result

Done.

---

## ⚡ Example

### Input:
john@test.com  
john@test.com  
   jane@test.com  

bob@test.com  

### Output:
john@test.com  
jane@test.com  
bob@test.com
